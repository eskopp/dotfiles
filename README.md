# dotfiles

Basic Wayland / Hyprland dotfiles for Arch Linux.

## Included
- Hyprland
- Hyprpaper
- Hyprlock
- Waybar
- Kitty
- Dunst
- Rofi
- Thunar

## Install
Run:
./install.sh

Then place a wallpaper here:
~/.config/wallpapers/default.png

Start Hyprland from a TTY with:
~/.local/bin/start-hyprland
